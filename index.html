西递

北宋 船形 三进院落

西递建村于北宋，以船形村落格局和三进院落著称。街巷尺度亲密，空间序列清晰， 体现徽州民居礼序与居住美学的平衡。

建村年代：北宋
格局特点：船形，三进院落
核心建筑：敬爱堂、追慕堂
空间气质：街巷紧凑、院落深进

结构拆解

选择左侧构件，查看对应的实拍图与拆解示意图

构件名称：马头墙（也称封火山墙、叠落山墙）
功能原理：1. 防火：高出屋面的墙体可阻断火势蔓延，避免连片民居火烧连营。
2. 防风：减少强风对屋面瓦件的破坏，增强建筑整体稳定性。
3. 排水：阶梯式坡面利于雨水快速排离，减少屋面渗漏。
4. 空间界定：形成高低错落的村落天际线，提升视觉层次与辨识度。
营造工艺：1. 墙体主体以青砖错缝砌筑，灰浆饱满，坚固耐久。
2. 顶部呈阶梯状层层叠落，常见三叠、五叠、七叠，等级越高叠数越多。
3. 墙面抹白灰，顶部压顶线条流畅，檐角微微上翘，形似昂首马头。
4. 大户人家会在墙头做简单线脚、雕花，增强装饰性。
文化寓意：1. 形似战马昂首，寓意“马到成功、仕途顺遂”。
2. 层层向上叠落，象征家族兴旺、步步高升。
3. 高低错落的形态也暗含风水“藏风聚气”的理念。

三雕艺术墙

木雕 · 渔樵耕读
雕种：木雕
典型位置：西递敬爱堂门窗隔扇、宏村承志堂厢房、南屏民居厅堂隔扇
图案内容：画面分为四组：渔夫垂钓、樵夫担柴、农夫犁田、书生伏案读书，配以山水、树木、云纹、山石，构图疏密得当，人物动态自然，场景完整连贯。
工艺技法：以透雕+深浮雕+线刻为主，多层镂空，层次可达3-5层，刀法圆润细腻，打磨光滑，保留木材本色，部分构件描金或彩绘。
文化寓意：体现“耕读传家”的核心价值观，农商并重、文武相济，表达对安稳、富足、知礼守义生活的追求。
木雕 · 松鹤延年
雕种：木雕
典型位置：宏村承志堂梁垫、月梁、檐枋；西递履福堂正厅横梁
图案内容：苍松挺拔、仙鹤伫立或展翅，搭配灵芝、祥云、寿石、溪泉，构图对称饱满，气势庄重。
工艺技法：深浮雕为主，局部镂空，木料多用樟木、杉木，防虫耐腐，线条流畅有力。
文化寓意：松象征长青不老，鹤象征长寿祥瑞，整体寓意长寿安康、家运绵长。
砖雕 · 福禄寿喜
雕种：砖雕
典型位置：西递、宏村普通民居门罩、门头上方
图案内容：蝙蝠（福）、梅花鹿（禄）、仙鹤（寿）、喜鹊（喜），搭配古钱纹、缠枝莲、祥云。
工艺技法：浅浮雕+线刻，青砖经水磨、精雕，线条简练干净，整体平整大气。
文化寓意：集福、禄、寿、喜四大吉祥寓意于一体，是徽州民居常见的吉祥纹样。
砖雕 · 八仙过海
雕种：砖雕
典型位置：呈坎罗氏宗祠、宏村祠堂隔扇门窗
图案内容：汉钟离、吕洞宾、张果老、韩湘子、铁拐李、曹国舅、蓝采和、何仙姑八位仙人各执法器，衣袂飘飘，神态各异。
工艺技法：透雕+圆雕结合，人物比例准确，衣纹飘逸，细节精致入微。
文化寓意：辟邪驱灾、平安顺遂、超凡脱俗，寄托家族兴旺、远离灾祸的愿望。
石雕 · 狮子戏球
雕种：石雕
典型位置：棠樾牌坊柱础、西递胡氏宗祠门墩、宏村祠堂入口
图案内容：石狮成对，雄狮戏球、雌狮抚幼狮，造型威严，肌肉饱满，神态生动。
工艺技法：圆雕+浅浮雕，石材选用青石、花岗岩，刀法刚劲有力，质感厚重。
文化寓意：镇宅辟邪、守护家宅、吉祥兴旺、权势尊严。
石雕 · 荷花漏窗
雕种：石雕
典型位置：宏村、西递民居院墙漏窗、天井花窗
图案内容：荷花、荷叶、莲蓬、藕节交错镂空，构图雅致简洁。
工艺技法：透雕+线刻，石材打磨光滑，镂空处棱角圆润。
文化寓意：“荷”通“和”，寓意家庭和睦、和和美美、清廉高洁。
1. 艺术特点：题材吉祥、构图饱满、刀法精湛、因材施艺。
2. 文化功能：装饰美化、教化育人、风水祈福、彰显身份。
3. 非遗地位：徽州三雕为国家级非物质文化遗产，是皖南民居灵魂所在。
4. 资料来源：安徽博物院、徽州文化博物馆、黄山市文旅局、非遗保护中心。

宏村水系动画

宏村牛形人工水系始建于南宋绍兴元年，由汪氏先祖汪思齐、胡重等人依据传统风水理论规划修建，历经数代完善成型。水系以“牛”为整体形态，集生活用水、农业灌溉、消防灭火、村落景观、微气候调节与风水堪舆于一体，是宏村世界文化遗产价值的重要核心。

水系核心布局（牛形对应）

牛头：雷岗山
牛角：村口两棵参天古树
牛身：宏村民居建筑群
牛肠：贯穿全村的人工水圳
牛胃：月沼
牛肚：南湖
排水系统：地下暗沟与雨污分流

水圳源头（西溪入口）

具体位置：宏村西北侧，源自黄山余脉羊栈岭山涧溪流。

水流方向：自西北向东南，贯穿整个村落。

核心功能：水源供给、水流调控、水质净化。

营造特点：位于高处，依地势生态自流，无需人工动力。

主水圳（牛肠）

具体位置：沿宏村街巷蜿蜒，分流至家家户户门前。

水流方向：自西北向东南，分支环绕民居院落。

核心功能：生活用水、消防用水、降温调湿、灌溉用水。

营造特点：宽约1米、深0.5-0.8米，青石砌筑、底铺鹅卵石，上游饮用下游洗涤。

月沼（牛胃）

具体位置：宏村中心，汪氏宗祠北侧。

水流方向：主水圳汇入，蓄水后向南流出至南湖。

核心功能：蓄水调流、生活核心、风水核心、消防储备。

营造特点：半月形青石围砌，体现“月盈则亏、凡事有余”的徽州哲学。

南湖（牛肚）

具体位置：宏村南侧，村落末端。

水流方向：月沼出水汇入南湖，多余水流外排农田。

核心功能：总蓄水、灌溉、景观休闲、防洪排涝、生态净化。

营造特点：弧形大水面，南湖书院与石拱桥相映，水利与人文景观高度融合。

地下排水系统

具体位置：街巷与民居天井下方，隐蔽式全程暗沟。

水流方向：自北向南，与地表水同向，汇入南湖。

核心功能：排污排水、防涝防潮、水质保护（雨污分流）。

营造特点：青石砌筑，不占地面空间，数百年运行仍通畅。
